# UrMechanic 🔧
## India's AI Vehicle Doctor

Gaadi ki problem? AI se poocho!

---

## What This App Does
- AI diagnoses car & bike problems from symptoms
- Shows parts prices in ₹ (OEM vs Aftermarket vs Used)
- Pros & cons for every part option
- Mechanic brief in English + Hindi
- Region-based delivery estimates across India
- UrMechanic Certified Mechanic network

---

## How to Deploy (Step by Step)

### Option 1 — Vercel (Recommended, Free)
1. Push this code to GitHub
2. Go to vercel.com → New Project
3. Import your GitHub repo
4. Click Deploy
5. Done! You get a free URL like urmechanic.vercel.app

### Option 2 — Run Locally
```bash
npm install
npm start
```
Open http://localhost:3000
